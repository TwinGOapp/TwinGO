# TwinGO

Twingo is an Albanian news portal that'll keep you up-to-date with the latest Albanian and international news. 

Using Twingo is really easy because it works just like any other blog: as soon as you start the app, you'll view all the news organized by uploading time. You can also filter the content by topics and look for news articles that include specific keywords using the search bar. 

You can find all kinds of content such as celebrity gossip, politics or the latest movies. If you ever get tired of reading, Twingo also offers its users the option of distressing by playing Rainbow Stacker, a game that's integrated into the app. What else could you ask for?
